# 🛡️ ExamGuard

ExamGuard is a browser-based examination management prototype developed as a B.Tech academic project. It is designed to simplify examination administration by providing dedicated portals for the **Head Administrator**, **Administrator**, and **Student** within a single web application.

The system demonstrates role-based access, examination scheduling, student management, attendance tracking, hall allocation, and an integrated examination interface. The prototype is built entirely using front-end technologies with browser storage, making it easy to run without a backend server.

# ✨ Features

## 👨‍💼 Head Administrator

* Secure Head Administrator login
* System dashboard
* Monitor examinations across administrators
* View attendance records
* View examination recordings
* Overall administrative oversight

---

## 👨‍💼 Administrator

### Hall Management

* Create and manage examination halls
* Generate hall access passwords
* Configure hall capacity

### Examination Management

* Create examinations
* Schedule examinations
* Set examination date and time
* Configure examination duration
* Assign halls to examinations

### Student Management

* Register students
* Manage student records
* Assign students to examination halls

### Attendance

* Mark and manage attendance
* View attendance records

### Recording Management

* Start recording (prototype)
* Stop recording (prototype)
* Playback recorded sessions

---

## 🎓 Student

* Secure student login
* View assigned examination hall
* View examination schedule
* Access examination interface
* Answer multiple-choice questions
* Live countdown timer
* Submit examination

---

# ⚙️ System Features

* Role-based access control
* Three dedicated user portals
